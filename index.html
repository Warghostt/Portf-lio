Projeto: WireFrame para Site de PET

Este projeto foi criado para explorar o design e funcionalidade de um site voltado para pet shops. Utilizei ferramentas como Figma para o design e prototipagem mas futuramente vou usar minhas habilidades e conhecimento em HTML/CSS/Bootstrap para criar um protótipo com maior imersividade com o projeto. A ideia principal foi desenvolver uma interface amigável para clientes acessarem produtos e serviços relacionados a pets.

Tecnologias usadas: Figma
